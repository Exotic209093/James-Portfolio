# GitHub Repository Setup

## Option 1: Create Repository via GitHub Website (Recommended)

1. Go to [GitHub.com](https://github.com) and sign in
2. Click the "+" icon in the top right â†’ "New repository"
3. Repository name: `portfolio-website` (or your preferred name)
4. Description: "Modern portfolio website built with Next.js 14, TypeScript, and Tailwind CSS"
5. Choose **Public** or **Private**
6. **DO NOT** initialize with README, .gitignore, or license (we already have these)
7. Click "Create repository"

## Option 2: Use GitHub CLI (if installed)

If you have GitHub CLI installed, run:
```bash
gh repo create portfolio-website --public --source=. --remote=origin --push
```

## After Creating the Repository

Once you've created the repository on GitHub, run these commands:

```bash
git remote add origin https://github.com/YOUR_USERNAME/portfolio-website.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` with your actual GitHub username.
